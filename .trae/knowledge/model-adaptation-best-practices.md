# 模型适配最佳实践

## 文件结构

```
model/
├── config.json
├── model.safetensors
├── tokenizer.json
└── tokenizer_config.json
```

## 适配流程

1. 环境检查 → 2. 模型分析 → 3. 算子映射 → 4. 量化配置 → 5. 部署测试

## 配置建议

| 场景 | TP | DP | 说明 |
|------|----|----|------|
| 低延迟 | 高 | 低 | 减少通信 |
| 高吞吐 | 中 | 高 | 增加批处理 |
| 内存受限 | 高 | 低 | 分散权重 |

## 测试要点

- 功能: 基本推理正常
- 精度: 损失<1%
- 性能: 达到预期指标
- 稳定: 无崩溃/内存泄漏
